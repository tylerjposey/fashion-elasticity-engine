# Modeling Assumptions

| Area | Assumption |
|---|---|
| Price history | Weekly prices are simulated because source data lacks complete markdown history |
| Elasticity | Log-log regression at SKU level |
| Inventory age | Approximated using weeks on market |
| Holding cost | $0.50 per unit per week |
| Discount search | 5% to 40% |
| Purpose | Decision-support prototype, not production pricing automation |
